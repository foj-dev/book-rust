## comparison

The act of checking if two or more items are equal, unequal, or ordered relative to one another.

```rust
// ==
let a = 1;
let b = 1;
println!("{}", a == b);

//
let a = 2;
let b = 3;
println!("{}", a == b);

// !=
let a = 1;
let b = 1;
println!("{}", a != b);

//
let a = 2;
let b = 3;
println!("{}", a != b);

// >
let a = 1;
let b = 1;
println!("{}", a > b);

// <
let a = 2;
let b = 3;
println!("{}", a < b);

// >=
let a = 1;
let b = 1;
println!("{}", a >= b);

// <=
let a = 2;
let b = 3;
println!("{}", a <= b);
```
